Built for the [Gemini 3 global hackathon](https://googledeepmind-gemini3.devpost.com/) on Devpost using Google's [AI Studio](https://aistudio.google.com/). Visual regulatory intelligence platform with five analysis modes: monitoring, simulation, comparison, roadmap generation, and trend analysis. Uses Gemini 3 Pro for reasoning-heavy tasks and Gemini 3 Flash for low-latency features, with Google Search grounding for real-time citations. Interactive global map built with D3.js and TopoJSON for drilling down to state-level jurisdictions.
